What is SceneSniffer?

SceneSniffer is a browser extension that uses AI to identify which TV show or movie a YouTube Short clip comes from. Never again will you struggle to find the source of that amazing scene you just watched!

One-Click Identification

Simply click the SceneSniffer button while watching a YouTube Short to instantly identify the source.

Detailed Results

Get show name, season, episode number, and even timestamp information to find exactly what you're looking for.

AI-Powered

Leveraging advanced computer vision and natural language processing to deliver accurate results.

Install the Demo

Try SceneSniffer right now by following these steps:

  1. Download the extension ZIP file
  2. Unzip the file to a folder on your computer
  3. Open Chrome and navigate to chrome://extensions
  4. Enable "Developer mode" using the toggle in the top-right corner
  5. Click the "Load unpacked" button and select the unzipped folder
  6. SceneSniffer should now appear in your extensions
  7. Go to YouTube Shorts and try it out!
